# TypeScript Advanced Assessment Writeup

## Overview
A TypeScript exercise covering advanced concepts such as generics, function overloads, utility types, and pure functions.

## What I implemented
- Generic constraints (Exercise 1)
- Utility (Exercise 2)
- Mapped Types (Exercise 3)
- Conditional types and infer (Exercise 4)
- unknown & User-Defined Type Guards (Exercise 5)
- Discriminated-Union Reducer (Exercise 6)
- Async Typing (Exercise 7)
- Literal Types (Exercise 8)
- Function Overloads (Exercise 9)
- Shopping cart system (Exercise 10)

## Key concepts used
- Type narrowing with `typeof`
- Function overloads
- Interfaces and type aliases
- Immutability (pure functions)
- Array methods like `map`, `filter`, and `reduce`

## Challenges faced
- File path issues when adding files to Git
- Understanding TypeScript overload syntax
- Ensuring functions remained pure (no mutation)

## What I learned
I learned how TypeScript enforces strict type safety and how to design scalable type-safe functions using generics and unions.

## Notes
All requirements were completed and tested using `npm run typecheck` and `npm test`.
